# PreOffice Start Center Branding

This directory contains the branding assets for the PreOffice Start Center.

## Assets

| File | Description |
|------|-------------|
| `logo.svg` | Main PreOffice logo (200x60) |
| `welcome-banner.svg` | Full-width header banner (800x120) |
| `app-writer.svg` | Writer application icon (64x64) |
| `app-calc.svg` | Calc application icon (64x64) |
| `app-impress.svg` | Impress application icon (64x64) |
| `app-draw.svg` | Draw application icon (64x64) |

## Configuration Files

| File | Description |
|------|-------------|
| `StartCenterConfig.xcu` | Product branding settings |
| `StartModuleConfig.xcu` | Start Center module commands |

## Installation

Copy assets to:
- `core/icon-themes/colibre/sfx2/startcenter/`

Copy configuration to:
- `core/officecfg/registry/data/org/openoffice/`

## Color Reference

- Primary: #2D8EFF (Presearch Blue)
- Secondary: #1A7AE8 (Darker Blue)
- Text: #333333
- Background: #FFFFFF
