# Trademark Guidelines

## Presearch Trademarks

The following are trademarks of Presearch:
- Presearch (word mark)
- Presearch logo (design mark)
- PreOffice (word mark)
- PreGPT (word mark)
- PreStorage (word mark)
- Pre-suite (word mark)

### Permitted Uses
- Refer to PreOffice by name in truthful statements
- Link to official Presearch websites
- Use in academic or journalistic contexts
- Use in compatible product descriptions ("Works with PreOffice")

### Prohibited Uses
- Implying official endorsement without permission
- Modifying Presearch logos
- Using marks in domain names without permission
- Creating confusingly similar marks

### Contact
For trademark inquiries: legal@presearch.com

## LibreOffice Trademarks

"LibreOffice" is a registered trademark of The Document Foundation.

### Our Usage
PreOffice uses the LibreOffice name only as required for attribution:
- "Based on LibreOffice technology" in About dialog
- License and copyright notices

We do NOT:
- Use "LibreOffice" in our product name
- Imply endorsement by The Document Foundation
- Use LibreOffice logos in our branding

### For More Information
See: https://wiki.documentfoundation.org/TradeMark_Policy

## Open Source Licensing vs Trademarks

Note that open source licenses (like MPL 2.0) grant rights to the code,
but do NOT grant trademark rights. You may:
- Fork and modify the code
- Distribute modified versions

You may NOT:
- Use our trademarks without permission
- Imply your fork is the official PreOffice

If you create a fork of PreOffice, please choose a different name.
